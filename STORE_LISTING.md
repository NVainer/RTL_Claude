## Name
Claude RTL — Hebrew & Arabic

## Summary (≤132 chars)
Right-to-left support for Hebrew & Arabic on claude.ai. Auto-directs each paragraph; code blocks stay left-to-right.

## Category
Productivity

## Detailed description
Claude RTL adds proper right-to-left (RTL) support for Hebrew and Arabic on
claude.ai.

Each paragraph is directed automatically by its first strong character, so your
Hebrew sentences align to the right even when they're packed with English
technical terms (KQL, Entra ID, RTL/LTR, etc.). The input box is handled too, so
what you type reads naturally as you write it.

Highlights:
• Per-paragraph direction — no awkward whole-page flip
• Works in the message area AND the input composer
• Code blocks always stay left-to-right, even with Hebrew comments
• One-click ON/OFF toggle in the toolbar; your choice is remembered
• Fully reversible — turning it off removes every change cleanly
• No data collection, no network requests — everything runs locally

Open source: https://github.com/NVainer/RTL_Claude
